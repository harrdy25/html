Position Property

Static Property

Elements Static are not Move any side

Element with static property can not move its Position of Left Right Top Bottom

Relative Property

Elements Static are Move any side

Element with static property can move its Position of Left Right Top Bottom

Absolute Property

Absolute Without Parents

Elements with Absolute Position are Move accroding to Body

Element move out from its normal flow

Element move with scroll




Absolute Property

Absolute With Parents

Elements with Absolute Position are Move accroding to Parents

Fixed Property

Elements with Fixed Position are Move accroding to Body and Can not scroll




Sticky Property

Relative + Fixed combination in this property

Initially work as a Relative, whenever thair condition(top: 10px) satisfied it work as a fixed
